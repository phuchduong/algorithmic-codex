# Metronic Dark Mode Implementation

This project demonstrates a functional dark mode toggle using the Metronic Tailwind HTML template. The mock UI focuses on seamless theme switching, allowing users to transition between light and dark visual styles dynamically.

## Project Overview

The goal of this implementation is to provide a clean, accessible interface that respects user preference through a dedicated theme switch. By leveraging Metronic's built-in CSS variables and utility classes, the site adapts its color palette, typography contrast, and component styling instantly.

## Key Features

* **Dynamic Theme Switching**: A toggle component that switches between 'light' and 'dark' modes.
* **Persistent Preference**: Logic to ensure the user's selected theme is maintained across sessions.
* **Tailwind Integration**: Utilizes Metronic's Tailwind CSS framework for rapid and consistent styling.

## Getting Started

1.  Clone this repository to your local machine.
2.  Ensure you have the Metronic assets installed in the appropriate directory.
3.  Open `index.html` in your browser to view the mock UI.
4.  Interact with the theme switch in the navigation bar to see the transition.
